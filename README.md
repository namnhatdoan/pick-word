# Pick Word 📖

An offline-first vocabulary practice app for kids. Parents manage a word library; children practice writing words daily via flashcards and challenges.

## Tech stack

- **React 18** — UI
- **Vite 5** — build tool & dev server  
- **vite-plugin-pwa** — Service Worker + Web App Manifest (installable, offline)
- **IndexedDB** — all data stored locally (words, sessions, test/written history)
